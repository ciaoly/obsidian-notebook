

# 字符串相关操作

#### 运算符:

|  运算符  | 描述                 | 备注                                                         |
| :------: | -------------------- | :----------------------------------------------------------- |
|    \+    | 连接两个字符串       |                                                              |
|    \*    | 按指定次数重复字符串 |                                                              |
|    -f    | 设置字符串格式       |                                                              |
| -replace | 替换运算符           | 用法："abcd" -replace "bc","TEST"  返回结果：aTESTd; 支持正则表达式 |
|  -match  | 正则表达式匹配       |                                                              |
|  -like   | 通配符匹配           |                                                              |
